Using jinja2, we can write loops inside HTML. With a little practice, you will get used to this.



Even knowing only a few pieces of jinja2 syntax will enable you to build a very interesting app.

Here is a jinja2 sample. Open the demo.html file (and look at lines 14-19) to see how it works.

DEMO CODE IS LIVE! The demo user is working on {{ list_name }}.

The user needs to study: {% for item in needs_work %} {{item}} {% endfor %}
